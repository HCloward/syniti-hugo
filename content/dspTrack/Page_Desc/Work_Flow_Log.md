# Workflow Log

<div class="use">

Use this page to [View Work Flow Messages
Sent](../Use_Cases/View_Workflow_Messages_Sent.htm).

</div>

To access this page, select **Track \> Configuration \> Work Flow \>
Work Flow Log**.

<td><p>TO</p></td>
