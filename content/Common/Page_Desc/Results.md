# Results

<div class="use">

Use this page to [View
Results](../../../Migration/Construct/Use_Cases/View_Results.htm).

</div>

To access this page:

1.  Click **Analyze** in *Navigation* pane.
2.  Click **Duplicates** for Data Source ID.
3.  Select an object.
4.  Click **Results** on Page toolbar.
