# Copy Position

<div class="use">

Use this page to [copy a position](../Use_Cases/Copy_a_Position.htm).

</div>

To access this page:

1.  Select <span style="font-weight: bold;">dspConduct
    \></span>**Security** in the *Navigation* pane.
2.  Select a position.
3.  Click the **Copy Position** icon.
