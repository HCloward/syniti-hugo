# Control (Formats)

<div class="use">

Use this page to [View Controls and Control
Formats](../../WebApp_Dev/View_Controls_and_Control_Formats).

</div>

To access this page:

1.  Select **Admin \> Configuration \> Setup \> Controls** from the
    *Navigation* pane.
2.  Click the **Formats** icon for a control.
