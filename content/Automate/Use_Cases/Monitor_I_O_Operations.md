# Monitor I/O Operations H

[Monitor I/O Operations V](#MonitorIOV)

To access this page:

1.  Select **IO Config \> File Operations** in the *Navigation* pane.
2.  Click **Vertical View** for the desired file operation.
3.  Click the **Monitor** button.
