# View Table Field Results

Results for profiled data sources can be viewed for individual fields
within a table.

To view fields:

1.  Click **Analyze** in the *Navigation* pane.
2.  Click **Profile** for Data Source ID.
3.  Click **Tables**.
4.  Click **Fields** for Table.
